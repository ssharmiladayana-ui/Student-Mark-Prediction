# Student Mark Prediction

### Aim
To predict student marks based on study hours using Linear Regression.

### Tech Stack
Python, Pandas, Scikit-learn, Matplotlib

### Dataset
Hours vs Scores with 25 records

### Result
Model achieved 95% accuracy with R2 Score = 0.95
For 9.5 hours study, predicted score is 93.

### How to Run
1. Open `student-mark-prediction.ipynb` in Google Colab
2. Run all cells
